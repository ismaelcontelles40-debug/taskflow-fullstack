# Nombre del proyecto

TaskFlow Fullstack

---

# Descripción general

TaskFlow Fullstack es una aplicación web fullstack diseñada para gestionar tareas y mejorar la productividad personal.

La aplicación permitirá crear, editar, completar y eliminar tareas mediante una interfaz moderna desarrollada con React, TypeScript y Tailwind CSS.

Además, contará con un backend desarrollado con Node.js y Express que actuará como API REST para gestionar la información de las tareas.

---

# Problema que intenta resolver

Muchas personas organizan sus tareas utilizando aplicaciones demasiado complejas o poco intuitivas.

Este proyecto busca ofrecer una solución sencilla, rápida y visual para gestionar tareas diarias desde cualquier dispositivo.

La aplicación permitirá tener una visión clara del trabajo pendiente y del progreso realizado.
